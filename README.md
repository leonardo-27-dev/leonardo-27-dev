### 🚀 Sobre Mim

Sou **Desenvolvedor Full Stack & Mobile** com forte experiência na criação de sistemas corporativos de alto desempenho, aplicativos multiplataforma e soluções embarcadas/TV. Meu foco está em projetar arquiteturas limpas (Clean Architecture), eficientes e escaláveis utilizando o ecossistema **.NET (C#)** e **TypeScript (NestJS)** no Backend, aliados a **React Native**, **Kotlin Multiplatform (KMP)** e **Android Nativo** no Mobile/TV.

- 💻 **Especialidades:** Clean Architecture, Domain-Driven Design (DDD), CQRS, Mensageria (RabbitMQ) e Integrações Cloud (AWS/Azure).
- 📱 **Mobile & TV:** Kotlin Multiplatform (KMP), Compose Multiplatform, Android TV nativo (MVVM, Koin, Coroutines), React Native (Expo).
- ⚙️ **Qualidade & DevOps:** Testes unitários/E2E (xUnit, Jest), CI/CD pipelines (Azure Pipelines), Docker e Git Workflow.
- 🌍 **Idiomas:** Português (nativo), Inglês (basico)

### 🔥 Principais Projetos & Engenharia Prática

* **🏨 Hotel Experience (HXP & HXP_AndroidTV)**  
  *Um ecossistema completo para o setor hoteleiro:*
  - **Backend corporativo em .NET (C#)** integrando serviços AWS (S3, Parameter Store), estruturado sob padrões DDD/CQRS, injeção de dependência e entrega contínua via Azure Pipelines.
  - **Aplicativo Android TV nativo** para hóspedes, permitindo assistir TV, realizar pedidos, acessar extrato de conta e apps parceiros. Desenvolvido em Kotlin com arquitetura MVVM, injeção de dependência via Koin, chamadas assíncronas com Retrofit e Coroutines, e decodificação/reprodução de mídia (FFmpeg/ExoPlayer).

* **🛒 Listick (Ecosistema de Listas)**  
  *Um ecossistema completo para gestão de compras offline-first:*
  - **Listick API (.NET 10 & PostgreSQL):** Desenvolvida em Clean Architecture com CQRS (EF Core + Dapper para leituras de alto desempenho), pipeline do MediatR com validações FluentValidation, controle de taxa de requisições, internacionalização automática (6 idiomas) e sincronização offline em lote utilizando resolução de conflitos baseada em carimbo de data/hora do cliente.
  - **Listick App (React Native & Expo):** Interface móvel em TypeScript e TailwindCSS (NativeWind), com persistência robusta para suportar operações offline eficientes.

* **🔑 Vega EasyCheckIn**  
  *Solução integrada de check-in para autoatendimento:*
  - **Backend Core em NestJS (TypeScript) & TypeORM** com suporte a múltiplos bancos de dados (SQL Server & PostgreSQL), monitoramento de logs do Sentry e documentação Swagger.
  - **API complementar em .NET Core** e **Interface de Totem Multiplataforma** desenvolvida em **Kotlin Multiplatform (KMP)** e **Compose Multiplatform** para displays interativos.
